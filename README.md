# 🐖 Detecció de Comportament i Estimació de Pes de Porcs (2D i 3D)

Aquest repositori conté el codi del meu Treball de Final de Grau (TFG) en Enginyeria Informàtica per a la Universitat Autònoma de Barcelona (UAB).

## 📖 Descripció
El projecte consisteix a crear un "pastor digital" per monitorar de forma autònoma el comportament, la salut i el pes dels porcs a les granges. S'utilitzen tècniques d'Intel·ligència Artificial i Visió per Computador per analitzar imatges 2D i mapes de profunditat 3D.

L'objectiu és oferir una eina que garanteixi el benestar animal i ajudi els ramaders a optimitzar el rendiment de la granja, sense la necessitat de fer una vigilància manual constant.

## 🎯 Objectius principals
* Identificar i fer seguiment de cada porc de forma individual.
* Detectar comportaments bàsics i possibles problemes de salut.
* Calcular el pes dels animals utilitzant càmeres 3D.
* Comparar l'eficàcia de diferents models de Visió per Computador (YOLO, CNN, etc.).

## 🛠️ Eines i Tecnologies
* **Llenguatge:** Python
* **IA / Visió per Computador:** YOLO, CNN, SAM
* **Organització:** GitHub i Trello (Metodologia Agile)
